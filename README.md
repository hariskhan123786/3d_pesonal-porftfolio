# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Go to project folder

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Preview build

```bash
npm run preview
```
